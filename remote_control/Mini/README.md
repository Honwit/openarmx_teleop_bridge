# Mini Remote Control

This directory stores OpenArm Mini based remote control projects for OpenArmX.

## Versions

- stable_v1
  - Current verified working version
  - Includes:
    - `openarmx_teleop_bridge/` ROS2 bridge package
    - `docs/` debugging summary, migration checklist, and SOP

## Recommended Usage

Use `stable_v1` as the baseline version for future changes.
When making new experiments, create new sibling folders such as:

- `stable_v2`
- `experiment_gripper_tuning`
- `experiment_vr_mapping`
